# Nursing Platform Lecturer Guide

## Access

- Open `/ng/nursing/login`.
- Use a seeded lecturer account from `docs/nursing-platform-demo-credentials.md`.
- Lecturer routes resolve to `/ng/nursing/lecturer`.

## Course Operations

Lecturers use the LMS tab to create courses, add lessons, and review course sections. Seeded courses include section metadata, lesson resources, enrollment records, and lesson progress data.

Assignments are created from the Assignments tab. Each assignment stores course, instructions, due date, rubric, maximum score, status, and creator metadata.

## Assessment Operations

The Gradebook tab exposes submitted work and a marking queue. Lecturers can enter a numeric score and feedback. Posted grades create gradebook records linked to the assignment submission.

The Quizzes, Simulation, Telehealth Lab, and Reports tabs provide review surfaces for quiz performance, case-based simulation activity, telehealth skill practice, and cohort-level reporting.

## Community Operations

Lecturers can post academic timeline announcements, comment on posts, reply to course Q&A threads, and guide student discussions. HOD and admin roles handle formal timeline moderation.
